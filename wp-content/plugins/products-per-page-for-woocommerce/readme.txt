Products per page for WooCommerce.

== Description ==

**Products per Page for WooCommerce** plugin lets you add *products per page selector* to the frontend of your WooCommerce store.

= Main Features =

* Multiple and customizable **frontend positions** (before and after shop loop).
* **Template options**: template, select class and style, before and after HTML.
* Option to **enable/disable cookie**.
* Customizable **select options** ([Pro version](https://wpfactory.com/item/products-per-page-woocommerce/)).

== Installation ==

1. Upload the entire plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Start by visiting plugin settings at "WooCommerce > Settings > Products per Page".
